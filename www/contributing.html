Contributing to the javaeetutorial project

This page provides information for people interested in contributing to the javaeetutorial project. You can contribute to javaeetutorial in one of three ways:

  • By contributing code
  • By writing documentation
  • By providing feedback and testing

Contributing code

The javaeetutorial project's example code is designed to show how to develop different types of enterprise applications using the Java EE technologies. We're looking for developers who have experience writing enterprise applications, and are familiar with the Java EE APIs, especially web tier technologies (JavaServer Faces, servlets), Enterprise JavaBeans, web services, and the Java Persistence API.

If you'd like to work on the example code of javaeetutorial, we'd ask that you first familiarize yourself with the project by:

  • Reading the online tutorial.
  • Subscribing to the dev@javaeetutorial.dev.java.net mailing list.
  • Joining javaeetutorial as an Observer.

Once you become part of the development process, you can then request the Developer role for javaeetutorial.

Contributing documentation

The tutorial content of the javaeetutorial project is written for Java EE developers ranging in level from beginner to advanced, and requires that the contributor have a good understanding of the Java EE platform and experience in producing technical documentation.

If you'd like to help write the tutorial content for javaeetutorial, we'd ask that you first familiarize yourself with the project by:

  • Reading the online tutorial.
  • Subscribing to the dev@javaeetutorial.dev.java.net mailing list.
  • Emailing the Project Owner with a brief description of:
    • Your interest in javaeetutorial
    • Your experience with the Java EE platform
    • Your experience developing technical content

Providing feedback and testing javaeetutorial releases

We welcome any users who would like to contribute to javaeetutorial by offering feedback on the example code and tutorial content, or who would like to help test our examples and content for milestone releases.

If you'd like to provide feedback and testing for javaeetutorial, please:

  • Read the online tutorial.
  • Subscribe to the dev@javaeetutorial.dev.java.net mailing list.
  • Join javaeetutorial as an Observer.
